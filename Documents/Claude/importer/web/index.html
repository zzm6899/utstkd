Features

Everything a photographer needs,
nothing they don't.

Built for the import-and-cull workflow. Keyboard-first, distraction-free.

⚡
Keyboard-speed culling
P to pick, X to reject, Shift+B for best-of. Rate 0–5 with number keys. Never touch a mouse.
🎯
Burst grouping
Consecutive shots are automatically grouped as bursts. Review the whole series, pick one, collapse the rest.
🔍
Duplicate detection
Already-imported files are detected by name and size. Re-inserting a card never creates duplicates.
📁
Smart folder structure
Choose from presets like YYYY/MM/DD or build your own with custom tokens. Always organized.
📡
FTP output
Simultaneously mirror imported files to a FTP/FTPS server. Great for remote backup or tethered workflows.
🔄
Auto-import
Insert a DCIM card and Culler automatically imports to your preset destination. No clicks required.
🌓
Exposure normalization
Auto-adjust exposure during export using your anchor photo as the reference. Consistent results, faster.
✅
Checksum verification
Every imported file is verified by checksum. Your images arrive bit-perfect, every time.
💾
Backup copy
Set a second destination and every import is mirrored there automatically. Built-in redundancy.
